Timestamp Microservice Project

This project is from FreeCodeCamp's Back End Certification curriculum. It takes a natural language date or unix timestamp and returns a JSON object with both forms of timestamp info.

Example Usage

https://boiling-plains-46995.herokuapp.com/November%2026,%201987 https://boiling-plains-46995.herokuapp.com/564904800

Example Output

{ "unix": 564904800, "natural": "November 26, 1987"}
